# isbn-scanner
## Description
- Scanner tool for Keio SFC Japanese Lab's Book Information Registration

## Usage
- `isbn_cam_windows.py`: Adapted for Windows PC
- `isbn_cam_mac.py`: Adapted for MacBook, removed `winsound` module
